# مؤسسة أدوال للمصاعد - ADAWAL Est.

موقع إلكتروني متكامل لطلب وصيانة المصاعد وفق الشروط السعودية.

## الميزات
- طلب مصعد جديد (معالج بـ 3 خطوات)
- طلب صيانة مصعد
- اختيار المواصفات والخيارات
- الشروط السعودية SASO / SBC
- متابعة حالة الطلب

## تشغيل المشروع محلياً

```bash
npm install
npm run dev
```

## النشر على Vercel

1. ارفع المشروع على GitHub
2. افتح [vercel.com](https://vercel.com) وسجّل دخول
3. اضغط **Add New Project** واختر المستودع
4. اضغط **Deploy** — سيتم النشر تلقائياً ✅
